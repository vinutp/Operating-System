## Getting Started

### Prerequisites
- GCC compiler (for C programs)
- Bash shell (for shell scripts)
- Linux/Unix environment

### Compilation & Execution

**For C programs:**
```bash
gcc -o output_name source_file.c
./output_name
```

**For Shell scripts:**
```bash
bash script_name.sh
# or
chmod +x script_name.sh
./script_name.sh
```

## Topics Covered

- **File System Operations** - Directory traversal, file handling
- **Pointers & Data Structures** - Dynamic memory allocation, linked lists
- **Shell Scripting** - Bash programming, automation
- **Process Scheduling** - Priority scheduling, SJF, Round Robin
- **Memory Management** - Allocation strategies, fragmentation
- **System Calls** - Process creation, management, synchronization
- **IPC** - Pipes, message queues, shared memory
